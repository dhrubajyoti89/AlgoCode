## Description
The palindromic numbers are the numbers whose reverse is equal to the original number. Given number of rows print a valid palindromic pyramid.

# Example
```
Input: 5
Output:
1
1 2 1
1 2 3 2 1 
1 2 3 4 3 2 1 
1 2 3 4 5 4 3 2 1
```
