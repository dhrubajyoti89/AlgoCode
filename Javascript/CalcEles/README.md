# Calculate Distinct Elements

#### Description:

To check the number of distinct elements of a Bit Array.

##### PseudoCode:

a[0] = S (modulo 2^31)
for i = 1 to N-1
    a[i] = a[i-1]*P+Q (modulo 2^31) 

#### Example:
Input: 3,1,1,1
Output: 3

Input: 4,2,1,3
Output: 11
