## Description

A program to perform the following with two complex numbers A, B:

-   addition, A + B
-   subtraction, A- B
-   multiplication, A * B
-   division, A / B
-   mod(A)
-   mod(B)

## Example

First input will be 1st number with real and imaginary part similarly second number.

Input:  
2 1  
5 6

Output:  
7.00+7.00i  
-3.00-5.00i  
4.00+17.00i  
0.26-0.11i  
2.24
7.81
