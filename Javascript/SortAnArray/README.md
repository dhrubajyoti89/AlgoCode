## Sort an Array

# Description
Write a program to sort the first half of an array in ascending order and the second half in descending order.

# Example
##### input :
N = 12
2 6 5 7 0 10 13 9 15 12 1 8


##### Result
0 2 5 6 7 10 15 13 12 9 8 1
