### Problem Statement
Find the sum of all numbers in a sequence, where nth term is defined as Tn=n^2-(n-1)^2
first line defines numbers of terms in sequence.
output the sequence and sum

#### Input
3

#### Output
Sequence: 1 3 5 
Sum: 9
