## Operations on SinglyLinkedList

###### Problem Statement
Create a SingleLinkedList and insert elements into it. Perform the following operations on the List.
1. Insert at the Beginning.
2. Insert at the End.
3. Insert at a particular position.
4. Delete at the Beginning.
5. Delete at the End.
6. Delete at a particular position.
7. Search for an element.

###### Input
Read from the user.

###### Output
Display the LinkedList with specified modifications.
