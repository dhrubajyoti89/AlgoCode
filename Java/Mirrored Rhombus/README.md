## Description
A rhombus is a shape which has all sides congruent, opposite sides parallel and diagonals crossing at 90 degrees. Given number of rows print a valid mirrored rhombus

# Example
```

Example
Input: 4
Output:

      ****
    ****
  ****
****
```
