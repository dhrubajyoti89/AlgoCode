####Description
This is a program to encode and decode morse code. 

Note: Morse decoding will return a string in all uppercase.

#### Example
Input:<br>
 This is a test!<br>
Output:<br>
- .... .. ... / .. ... / .- / - . ... - -.-.--<br>

Input:<br>
- .... .. ... / .. ... / .- / - . ... - -.-.--<br>
Output<br>
THIS IS A TEST!<br>
