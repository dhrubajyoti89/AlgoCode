## HarshadNumber or not

###### Problem Statement
In mathematics, a Harshad number is a number that is divisible by the sum of its digits.

Example: input - 111
It is divisible by 3, sum of it's digits(1+1+1).

###### Input
Read from the user.

###### Output
State whether the given number is Harshad or not.
