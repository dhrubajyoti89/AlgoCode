<h1 align=center>Merge Two Sorted Lists</h1>

## Problem Statement
Given 2 sorted linked lists, merge the lists to a single sorted linked list.

### Sample Input
List1: 2 -> 4 -> 5 -> 6 -> 8 -> 9
List2: 1 -> 3 -> 7

### Sample Output
1 -> 2 -> 3 -> 4 -> 5 -> 6 -> 7 -> 8 -> 9
