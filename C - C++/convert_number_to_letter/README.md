# Problem Statement
Given a number create and display its word form.

### Constraints
Upper Limit: 9,99,999

### Sample Input 1
1234

### Sample Output 1
one thousand two hundred and thirty four

### Sample Input 2 
1,07,654

### Sample Output 2 
one lahk seven thousand six hundred and fifty four
