# Problem Statement
Find the number of ways that n students can occupy r seats in a classroom

### Constraints
Number of students must be greater than the number of seats 

### Sample Input 1
9
5

### Sample Output 1
15120

### Sample Input 2 
6
3

### Sample Output 2 
120
