# Problem Statement

### Description
Given all sides, calculate the triangle area and print the sides in ascending order of their areas.

### Example
First line will be the number of test cases.
Input:
3
7 24 25
5 12 13
3 4 5

Output:
3 4 5
5 12 13
7 24 25
