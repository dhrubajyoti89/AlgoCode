<h1>Infix to Postfix</h1>
<h2>Description</h2>

Given an expression in the infix notation, this program displays the postfix equivalent of that expression. Supported operators are (`^`, `*`, `%`, `/`, `+`, `-`)
<br>
<h2>Example</h2>

Input: (P%Q)/A*B^C+(R-S%U)+W^Z

Output: PQ%A/BC^*RSU%-+WZ^+
